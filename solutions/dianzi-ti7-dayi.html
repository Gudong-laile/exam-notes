题7 动态分析 三个疑问解答

你那道题做得很标准,我针对你问的三个点解释清楚。

问题1: 集电极和发射极之间那个"奇怪的小菱形"是什么?

那个菱形符号是受控电流源 (Controlled Current Source) 的标准电路符号。

为什么用菱形?

电路图里有两种"源":

符号含义
圆圈 ⭕独立源 (Independent source) — 电压/电流由自己决定
菱形 ◇受控源 (Dependent source) — 电压/电流由别处的物理量控制

晶体管的核心特性就是: 集电极电流 $i_c$ 由基极电流 $i_b$ 控制,关系是 $i_c = \beta i_b$。

这不是一个独立的电源,而是"被 $i_b$ 控制"的电流源,所以用菱形画。

物理意义

晶体管放大的本质就是: 你在 B-E 之间送一个小电流 $i_b$,在 C-E 之间就会"凭空"出现一个 $\beta$ 倍大的电流 $\beta i_b$。这个被"凭空"产生的大电流就是用菱形受控源表示的。

记忆口诀: 菱形 = 听话的源,看别人脸色干活。

问题2: 输出电压为什么要加负号?

这是共射放大电路的反相特性,需要从电流方向理解。

推导过程

看你的微变等效电路:

  • 受控源 $\beta i_b$ 的箭头从 C 指向 E (向下)
  • 这意味着: 电流从外电路流入 C 节点的方向是 C → E 内部 → 地
  • 等价地说,外电路上,电流是从地(E)通过 $R_C \parallel R_L$ 流向 C 的方向

输出电压 $u_o$ 的定义

题目里 $u_o$ 是集电极到地的电压(C 点高电位,地为参考)。

按 $u_o$ 的参考方向(C 为 + 端,地为 - 端),电流在 $R_C \parallel R_L$ 上的实际流向是从地流向 C。

这意味着: 电流方向与 $u_o$ 的参考方向相反。

所以:

$$u_o = -\beta i_b \cdot (R_C \parallel R_L)$$

直观理解

更直观的物理理解:

  • 当 $u_i$ 上升 → $i_b$ 增大 → $i_c$ 增大 (更多电流从 C 流入晶体管)
  • $R_C$ 上的电压降 $i_c \cdot R_C$ 增大
  • 因为 $u_C = V_{CC} - i_c R_C$,所以 $u_C$ 下降
  • $u_o$ 取自 C 点,所以 $u_o$ 下降

结论: $u_i$ 升,$u_o$ 降 → 反相 → 加负号。

记忆口诀

共射放大有"两负":

  1. 电流方向与 $u_o$ 参考方向相反
  2. 波形反相 (输入升 = 输出降)

这两个"负"是同一件事的两种表述。

问题3: $r_{be}$ 极小、$r_{ce}$ 极大,为什么?

你的直觉完全正确! 这正是分析时为什么把 $r_{ce}$ 忽略掉、只算 $r_{be}$ 的原因。

$r_{be}$ 为什么小?

$r_{be}$ 是 B-E 结的正向偏置等效电阻。

  • 在放大区,B-E 结是正偏的 PN 结
  • PN 结正偏时导通良好,电阻很小
  • 按你教材公式: $r_{be} \approx 200 + (1+\beta)\dfrac{26}{I_{EQ}}$
  • 典型值: 当 $\beta=50$,$I_{EQ}=1\text{mA}$ 时,$r_{be} \approx 1.5\text{k}\Omega$,千欧级

$r_{ce}$ 为什么大?

$r_{ce}$ 是 C-E 之间的输出电阻 (从 C 看进去的等效电阻)。

  • 在放大区,C-B 结是反向偏置的 PN 结
  • PN 结反偏时几乎不导电,电阻极大
  • 典型值: $r_{ce}$ 在 几十千欧 ~ 几百千欧 量级
  • 远大于 $R_C$ (通常 $R_C$ 是 1-10 kΩ)

为什么微变等效电路里没画 $r_{ce}$?

因为 $r_{ce}$ 是和 $R_C$ 并联的,而且 $r_{ce} \gg R_C$:

$$r_{ce} \parallel R_C \approx R_C$$

(大数和小数并联,结果≈小数)

所以忽略 $r_{ce}$ 几乎不影响结果,简化分析时直接当不存在。

总结表

参数大小物理原因是否要算
$r_{be}$小 (kΩ)B-E 正偏 PN 结必算
$r_{ce}$大 (几十~几百 kΩ)C-B 反偏 PN 结通常忽略

三极管三个 PN 结状态的对应关系

回顾你笔记里抄的三种工作状态:

  • 截止: $u_{BE}<0.5\text{V}$,B-E 反偏 → $r_{be} = \infty$ (不导通)
  • 放大: $u_{BE}\approx 0.7\text{V}$ 且 $u_{CE}>0.7\text{V}$,B-E 正偏 + B-C 反偏 → $r_{be}$ 小,$r_{ce}$ 大 ← 就是动态分析的工作区
  • 饱和: $u_{BE}\approx 0.7\text{V}$ 且 $u_{CE}\leq 0.3\text{V}$,两个结都正偏 → 两个电阻都小,失去放大能力

一句话总结

  • 菱形 = 受控源,代表晶体管"小电流控大电流"的本质
  • 负号 = 共射反相,因为电流方向与 $u_o$ 参考方向相反 / 物理上输入升输出降
  • $r_{be}$ 小 / $r_{ce}$ 大 = B-E 正偏 / C-B 反偏的 PN 结特性,所以分析时只算 $r_{be}$
