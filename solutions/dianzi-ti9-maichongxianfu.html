题9: 矩形脉冲输入的二极管限幅电路

输入: 矩形脉冲序列 (3V, 7V, 10V, -2V), $E=5\text{V}$, 忽略二极管正向压降. 求输出波形.

一、电路分析

拓扑

  • D: 阳极在输入端, 阴极在输出节点 (D 指向右)
  • $R$ 和 $E=5\text{V}$ 串联接到输出节点和地之间, $E$ 的正极朝上
  • $u_o$ 是输出节点对地电压

跟题8的关系

完全一样的电路, 只是:

  • 输入从正弦波 → 矩形脉冲
  • $E$ 从 3V → 5V

所以结论也一样: $u_o = \max(u_i, E) = \max(u_i, 5\text{V})$

二、判断 D 状态

设 D 截止: 无电流, $u_R = 0$, $u_o = E = 5\text{V}$

$$V_{\text{阳}} = u_i, \quad V_{\text{阴}} = u_o = 5\text{V}$$

D 导通条件: $V_{\text{阳}} > V_{\text{阴}}$, 即:

$$\boxed{u_i > 5\text{V}}$$

三、分段计算每个脉冲

按输入序列逐个分析:

输入 $u_i$判断D 状态输出 $u_o$
3 V3 < 5截止5 V (被 E 拉到 5V)
7 V7 > 5导通7 V (D 短路, 跟随输入)
10 V10 > 5导通10 V (D 短路, 跟随输入)
-2 V-2 < 5截止5 V (被 E 拉到 5V)

四、输出波形

u_o (V)
 10│            ┌───┐                    ← 10V 脉冲原样
   │            │   │
  7│      ┌───┐ │   │                    ← 7V 脉冲原样
   │      │   │ │   │
  5│──────┘   └─┘   └────────────────    ← 5V 削平线(D 截止时)
   │
  3│                                      ← 输入 3V 位置 → 输出 5V (削掉了 3V 那段)
   │
  0└──────────────────────────────── t
   │
 -2│                                      ← 输入 -2V 位置 → 输出 5V (削掉了 -2V 那段)

关键变化 (跟输入对比):

  • 输入 3V 的位置 → 输出变成 5V
  • 输入 -2V 的位置 → 输出变成 5V
  • 输入 7V → 输出 7V (原样)
  • 输入 10V → 输出 10V (原样)

输出序列: 5V → 7V → 10V → 5V

五、易错点 ⚠️

错误1: 忘记考虑 E 的存在

很多人看到"忽略正向压降"就以为 $u_o = u_i$ 永远成立. 错! E 才是关键的"地板", D 截止时 $u_o = E$, 不是 0.

错误2: 负输入直接当 0 算

输入 -2V 时, 输出不是 0V, 是 5V (因为 D 截止, E 把输出拉到 5V).

错误3: 把脉冲宽度搞乱

矩形脉冲是离散的, 每段都要单独判断 D 状态, 不要拿正弦波的连续逻辑套.

错误4: 忘记"忽略正向压降"的意思

"忽略正向压降" = 假设 D 导通时压降为 0, 即理想二极管. 不忽略的话, D 导通时 $u_o = u_i - 0.7\text{V}$, 所有导通段全部下移 0.7V.

七、考试答题模板

答:

设 D 截止, 则 $u_o = E = 5\text{V}$. D 导通条件为 $u_i > 5\text{V}$.

逐段分析:

  • $u_i = 3\text{V}$: $3 < 5$, D 截止, $u_o = 5\text{V}$
  • $u_i = 7\text{V}$: $7 > 5$, D 导通, $u_o = 7\text{V}$
  • $u_i = 10\text{V}$: $10 > 5$, D 导通, $u_o = 10\text{V}$
  • $u_i = -2\text{V}$: $-2 < 5$, D 截止, $u_o = 5\text{V}$

输出波形为: 所有低于 5V 的输入被削平在 5V, 高于 5V 的输入原样输出.

输出序列: 5V → 7V → 10V → 5V
