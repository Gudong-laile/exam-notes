题16: JK 触发器的四种工作状态

一、JK 触发器是什么?

JK 触发器(JK Flip-Flop) 是数字电路里最常用的双稳态触发器之一. 有两个控制输入 $J$ 和 $K$, 一个时钟输入 $CP$, 两个互补输出 $Q$ 和 $\overline{Q}$.

它的特点: 没有非法状态 — 不管 $J$、$K$ 怎么组合, 输出都有明确定义. 这是它比 RS 触发器(有"约束态")更好用的原因.

二、四种工作状态 (核心)

JK 触发器的输出由 $J$、$K$ 的四种组合决定. 设当前状态为 $Q^n$ (next state 为 $Q^{n+1}$):

状态 1: $J=0, K=0$ → 保持 (Hold)

$$Q^{n+1} = Q^n$$
  • 物理意义: 不管时钟来不来, 输出保持不变
  • 用途: 锁住当前数据

状态 2: $J=0, K=1$ → 置 0 (Reset)

$$Q^{n+1} = 0$$
  • 物理意义: 时钟来了, $Q$ 强制变成 0
  • 用途: 清零

状态 3: $J=1, K=0$ → 置 1 (Set)

$$Q^{n+1} = 1$$
  • 物理意义: 时钟来了, $Q$ 强制变成 1
  • 用途: 置位

状态 4: $J=1, K=1$ → 翻转 (Toggle)

$$Q^{n+1} = \overline{Q^n}$$
  • 物理意义: 时钟来了, $Q$ 翻转(0 变 1, 1 变 0)
  • 用途: 二分频、计数

三、特性方程 (一行总结)

把四种状态合写成一个布尔表达式 — JK 触发器的特性方程:

$$\boxed{Q^{n+1} = J \cdot \overline{Q^n} + \overline{K} \cdot Q^n}$$

可以代入四种情况验证:

$J$$K$代入特性方程化简状态名
00$0 \cdot \overline{Q^n} + 1 \cdot Q^n$$Q^n$保持
01$0 \cdot \overline{Q^n} + 0 \cdot Q^n$$0$置 0
10$1 \cdot \overline{Q^n} + 1 \cdot Q^n$$1$置 1
11$1 \cdot \overline{Q^n} + 0 \cdot Q^n$$\overline{Q^n}$翻转

记忆神器: 特性方程记住一个, 四种状态自动推出来.

四、状态转换表 (Truth Table)

记忆口诀

或者更口语化:

K 是 1 → 偏向 0; J 是 1 → 偏向 1; 两个都 1 → 翻; 都 0 → 不动

五、关键概念辨析

CP 边沿的重要性

JK 触发器的输出只在时钟 CP 的有效边沿才更新(通常是上升沿↑或下降沿↓, 看具体型号). CP 没有边沿时, 不管 J、K 怎么变, 输出都不变.

跟 RS、D、T 触发器的对比

类型输入数特性方程有无非法态
RS2 (R, S)$Q^{n+1} = S + \overline{R}Q^n$, R·S=0 约束有 (R=S=1)
JK2 (J, K)$Q^{n+1} = J\overline{Q^n} + \overline{K}Q^n$无
D1 (D)$Q^{n+1} = D$无
T1 (T)$Q^{n+1} = T \oplus Q^n$无

JK 触发器最大的优势: $J=K=1$ 不是非法态, 而是有用的"翻转态" — 这就是计数器的核心.

JK 和 T 的关系

当 $J = K = T$ 时, JK 退化成 T 触发器:

  • $T=0$: 保持(对应 J=K=0)
  • $T=1$: 翻转(对应 J=K=1)

所以 JK 触发器可以当成 T 触发器用, 把两个输入短接就行.

六、典型应用举例

应用1: 二分频电路 (J=K=1)

把 $J$ 和 $K$ 都接高电平(1), 输入时钟 $CP$:

  • 每来一个 CP 上升沿, $Q$ 翻转一次
  • 输出频率 = 输入频率 / 2

这是数字钟、分频器、计数器的基础.

应用2: 同步计数器

多个 JK 触发器串联, 每级触发器输出连到下一级的时钟, 就能构成 N 位二进制计数器 (模 $2^N$).

应用3: 寄存器、移位寄存器

利用置 0/置 1/保持的功能, 实现数据存储和移位.

七、考试答题模板

答: JK 触发器有四种工作状态, 由控制输入 $J$、$K$ 决定:

  1. 保持态 ($J=0, K=0$): $Q^{n+1} = Q^n$, 输出维持不变;
  2. 置 0 态 ($J=0, K=1$): $Q^{n+1} = 0$, 输出强制为 0;
  3. 置 1 态 ($J=1, K=0$): $Q^{n+1} = 1$, 输出强制为 1;
  4. 翻转态 ($J=1, K=1$): $Q^{n+1} = \overline{Q^n}$, 输出在每个时钟边沿翻转一次.

特性方程为 $Q^{n+1} = J \cdot \overline{Q^n} + \overline{K} \cdot Q^n$.

输出仅在时钟 $CP$ 的有效边沿更新.

八、易错点 ⚠️

错误1: 把 J=0, K=1 和 J=1, K=0 搞反

记住: K 像 "Kill" (杀 = 置 0), J 像 "Jump" up (跳起 = 置 1).

或者: J 控制 "Jump 到 1", K 控制 "Kill 到 0".

错误2: 漏掉 CP 边沿要求

如果题目只给 J、K 不给 CP, 不能直接说输出多少 — 必须等 CP 边沿. 没边沿就是保持上一个状态.

错误3: 跟 RS 触发器混淆

RS: S=1, R=1 是非法态 (禁止), R=1 是 Reset, S=1 是 Set.
JK: J=1, K=1 是翻转态 (合法), K=1 是 "Kill"(置 0), J=1 是 "Jump"(置 1).

JK 是 RS 的"升级版", 把非法态变成了有用的翻转态.

九、一句话总结

JK 触发器: 两个输入决定四种状态(保持/置 0/置 1/翻转), 特性方程 $Q^{n+1} = J\overline{Q^n} + \overline{K}Q^n$. 全靠 CP 边沿驱动.

最大亮点: $J=K=1$ 是翻转态 — 计数器的灵魂 😉
