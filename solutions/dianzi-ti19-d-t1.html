题19 D 触发器 t1 时刻输出

题目

电路:一个 D 锁存器,D 输入端通过外部连线接到 Q̄(反相输出),即 D = Q̄(交叉反馈接法);时钟输入标记为 C,直接接入(C 旁没有三角形 ▷,也没有小圆圈)。Q 引出为外部输出。

波形图:C 在 t1 时刻由低电平跳变为高电平,随后维持一段高电平,再变低,再变高……(矩形脉冲序列)。

触发方式判断:C 输入端仅为直接连线,无 ▷ 符号、无小圆圈 → 这是电平触发(电平敏感锁存器),且为"高电平触发"(C 为高电平期间锁存器透明,D 直接传到 Q;C 为低电平期间锁存器保持)。

(上次解答误将其当作边沿触发器,只看 t1 这一瞬间的跳变锁存一次 D 值,这是错误的——本题没有 ▷ 边沿符号,不能套用边沿触发的规则。)

解答

第一步,明确触发规则:
这是电平触发 D 锁存器,且无反相小圆圈,故为高电平触发:

  • 当 C = 1(高电平)期间:锁存器处于"透明"状态,输出 Q 跟随输入 D 的变化(D 怎么变,Q 就怎么跟着变,不是只采样一次)。
  • 当 C = 0(低电平)期间:锁存器处于"保持"状态,Q 锁定不变。

第二步,代入本题的反馈结构 D = Q̄:
由于 D 始终接的是本级输出的反相端 Q̄,在 C 维持高电平的整段时间内:

  • Q 跟随 D = Q̄ 变化 → Q 变为 Q̄(原值) → 而 Q 一变,Q̄ 立刻又变 → D 又变 → Q 又跟着变……
  • 这就形成了"空翻"(racing/振荡):只要 C 保持高电平,Q 和 Q̄ 会持续互相翻转,理论上以电路延迟为周期振荡,输出不稳定、不可预测(这是电平触发锁存器接成 D=Q̄ 时的经典反例,用来说明它不能像边沿触发器那样可靠分频)。

第三步,看 t1 时刻具体取值:
设 C 跳变为高电平之前(即 t1 之前,C=0 保持阶段)锁存器原状态为 Q=0(教材通常默认初态 Q=0,Q̄=1)。

t1 正是 C 由 0 跳变为 1 的那个时刻——也就是锁存器从"保持"刚刚进入"透明"状态的瞬间。在这一瞬间:

  • D = Q̄ = 1(取的是跳变前一瞬间的旧状态 Q̄)
  • 锁存器开始透明,Q 立即跟随 D,所以 Q(t1) 由 0 翻转为 1。

之后只要 C 维持高电平,Q 会继续在 1 和 0 之间空翻振荡,无法给出确定值;但题目只问 t1 这一时刻(即刚跳变进入高电平的那一刻),取的是第一次翻转后的值:

$$Q(t_1) = \overline{Q(t_1^-)} = \overline{0} = 1$$

答案

$$\boxed{Q(t_1) = 1}$$

(说明:t1 之后,由于 D=Q̄ 的交叉反馈和电平触发的"透明"特性,Q 会持续空翻振荡,不是稳定值;题目只要求 t1 时刻这一翻转结果。)

易错点

  1. 核心错误:把电平触发当成边沿触发。
    本题 CP(C)输入端没有三角形 ▷ 符号,说明它不是边沿触发器,不能用"只在上升沿/下降沿那一瞬间采样一次 D"的规则去分析。
  2. 正确识别方法:
    - 直接连线、无 ▷、无圆圈 → 电平触发,高电平有效(C=1 时透明,C=0 时保持)。
    - 直接连线、无 ▷、有圆圈 → 电平触发,低电平有效。
    - 有 ▷、无圆圈 → 边沿触发,上升沿有效。
    - 有 ▷、有圆圈 → 边沿触发,下降沿有效。
  3. 电平触发 + D=Q̄ 反馈 → 空翻问题。
    这是电平触发锁存器的经典缺陷:只要 CP 保持有效电平,输出会因反馈不断翻转,无法稳定计数/分频,这也是边沿触发器(主从结构、维阻结构)被发明出来取代简单电平锁存器的根本原因。
  4. t1 取的是"刚进入有效电平瞬间"的翻转后结果,而不是"整个高电平期间"的某个确定值——后者本身是振荡的、不确定的。

一句话总结

C 端无 ▷ 符号、无小圆圈,说明这是高电平触发的 D 锁存器,而非边沿触发;在 D=Q̄ 的交叉反馈下,C 进入高电平的瞬间(t1)锁存器输出由原状态翻转一次,故 $Q(t_1)=1$,此后还会因"空翻"持续振荡。
