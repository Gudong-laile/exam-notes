题19 D 触发器 t1 时刻输出

题目

电路为单个 D 触发器:

  • D 端通过外部连线接到 Q̄ 输出端,即 D = Q̄(反馈接法)
  • CP 端接时钟信号 C
  • 输出取 Q 端

C 波形为方波,从低电平开始,在 t1 时刻产生第一个上升沿(0→1 跳变),之后周期性出现高低电平交替的脉冲。

设触发器为边沿 D 触发器,在 CP 上升沿动作(由波形标注 t1 对应第一个上升沿可判断为上升沿触发)。

初始状态设 Q(0) = 0。

解答

D 触发器特性方程:

$$Q^{n+1} = D$$

本电路中 D 接 Q̄,代入得:

$$Q^{n+1} = \bar{Q}^{n}$$

即每来一个有效触发沿(上升沿),Q 翻转一次(这是用 D 触发器构成的 T' 触发器/二分频电路)。

逐沿分析:

  • 初始: Q(0) = 0,此时 D = Q̄(0) = 1
  • t1 时刻(第一个上升沿)触发: Q 更新为触发前的 D 值

$$Q(t_1) = Q^{n+1} = D(t_0^-) = \bar{Q}(0) = \bar{0} = 1$$

即在 t1 这个上升沿到来的瞬间,触发器按沿前一刻 D = Q̄ 的值翻转,Q 由 0 跳变为 1。

答案

$$\boxed{Q(t_1) = 1}$$

易错点

  1. 判断触发沿方向:必须看清是上升沿触发还是下降沿触发的 D 触发器(边沿触发器一般默认上升沿,但要结合波形标注确认 t1 对应的是哪类跳变),方向判断错会导致结果整体错位。
  2. D 接 Q̄ 还是接 Q:看清反馈线具体连的是 Q 端还是 Q̄ 端,本题接的是 Q̄,构成翻转(分频)电路;如果接的是 Q,则 Q 保持初值不变,结果完全不同。
  3. 初始状态假设:题目未明确给出初始 Q 值时,通常默认 Q(0) = 0,要在解答中明确写出这一假设,不能默认读者知道。

一句话总结

D 接 Q̄ 的 D 触发器等效于 T' 触发器,每个有效触发沿 Q 翻转一次,初始 Q=0 时第一个上升沿(t1)到来后 Q 变为 1。
