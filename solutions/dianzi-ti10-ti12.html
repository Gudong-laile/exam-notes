数电 题10 + 题12 合解

题10

题目描述

某产品有 A、B、C 三项质量指标,其中 A 为主要指标,B、C 为次要指标,产品检验标准规定:当主要指标 A 和至少一项次要指标都合格时,产品才定为合格品。试画出能检验出合格品的逻辑电路。

解答

设 A、B、C 为输入变量,取值 1 表示该指标合格,0 表示不合格。设 F 为输出,F=1 表示产品合格。

第一步:列真值表

"A 合格"且"B、C 中至少一项合格"才算合格品。

ABCF

1111

第二步:写逻辑表达式

由真值表可直接看出:只要 A=1,且 B、C 至少一个为 1,F 就是 1。即

F = A·(B+C)

(可用真值表逐项验证,也可直接由题意写出,无需化简,本身已是最简形式。)

第三步:转换为与非门实现

数字电路常要求全部用与非门(NAND)实现。利用两次取反不变(摩根定律):

F = A·(B+C) = \overline{\overline{A·(B+C)}}

把 (B+C) 也用与非门表示:利用 B+C = \overline{\overline{B}·\overline{C}},先用两个非门(可用与非门的两输入接同一信号实现,即 \overline{B} = \overline{B·B})得到 \overline{B}、\overline{C},再用一个与非门求 \overline{\overline{B}·\overline{C}} = B+C。

具体级联(全部用与非门,标号 G1~G4):

  • G1:输入 B、B → 输出 \overline{B}
  • G2:输入 C、C → 输出 \overline{C}
  • G3:输入 \overline{B}、\overline{C} → 输出 \overline{\overline{B}·\overline{C}} = B+C
  • G4:输入 A、(B+C) 这两路 → 先把 G3 输出取反送入与非门,即 G4 输入为 A 与 \overline{B+C} ...

注意这里不能直接套,需要重新理顺:与非门只能算"先与再非"。要得到 F = A·(B+C),做法是:

  1. 用 G3(与非门,B、C 取反后相与再取反,即上面三步)先算出 S = B+C。
  2. 再用一个与非门,输入 A 和 S,输出 \overline{A·S}。
  3. 最后再用一个非门(与非门两输入接同一信号)对 \overline{A·S} 取反,得到 F = A·S = A·(B+C)。

电路连接顺序(共 5 个与非门)

B ──┬──[NAND]── B̄ ──┐
    └──┘            │
                    [NAND]── (B+C) ──┐
C ──┬──[NAND]── C̄ ──┘                │
    └──┘                            │
                                    [NAND]── A·(B+C)的反 ──[NAND]── F
A ──────────────────────────────────┘         (两输入接同一信号取反)

即:G1、G2 分别把 B、C 自身取反(两输入端接同一根线);G3 把 \overline{B}、\overline{C} 相与再取反得到 B+C;G4 把 A 与 G3 的输出相与再取反;G5 把 G4 的输出自身取反(两输入端接同一根线),最终输出 F。

答案

$$\boxed{F = A\cdot(B+C)}$$

用 5 个二输入与非门级联实现(取反—或—与三层,每层用与非门加非门复合而成)。

易错点

  • 不要把"至少一项次要指标合格"误理解为"B 和 C 都合格"(那是与,不是或)。题目说的是"至少一项",对应 B+C,不是 B·C。
  • 全用与非门实现时,容易漏掉最外层的非门:G4 算出的是 \overline{A·(B+C)},必须再加一级取反才是 F 本身,不能直接把 G4 输出当作答案。
  • 用与非门实现"非"运算时,记得两个输入端接同一信号,即 \overline{X} = \overline{X\cdot X},不要漏画或漏接其中一个输入。

题12

题目描述

某羽毛球考试有三名裁判 A、B、C,其中 A 为主裁判,B 和 C 为副裁判。规则:只要主裁判 A 认为合格,考试就通过;若 A 认为不合格,则需要 B 和 C 均认为合格才能通过。试用与非门设计逻辑电路实现该功能。

解答

设 A、B、C 为输入,取值 1 表示该裁判认为合格,0 表示不合格;F 为输出,F=1 表示考试通过。

第一步:按题意列逻辑表达式

题目分两种情况:

  • A=1(主裁认为合格):直接通过,与 B、C 无关。
  • A=0(主裁认为不合格):必须 B 和 C 都合格(B·C=1)才能通过。

写成表达式:

F = A + \overline{A}\cdot B\cdot C

第二步:化简

利用吸收律 X + \overline{X}Y = X + Y:

F = A + \overline{A}\cdot B\cdot C = A + B\cdot C

验证真值表(部分关键行):

ABCF=A+BC
1001
0111
0100
0010

与题意一致:A=1 时无论 B、C 都通过;A=0 时只有 B、C 都为 1 才通过。

第三步:转换为与非门实现

F = A + B·C,先把 B·C 用与非门求两次反得到:

  • G1(与非门):输入 B、C → 输出 \overline{B\cdot C}
  • G2(非门,与非门两输入接同一信号):输入 \overline{B\cdot C} → 输出 B·C

再用摩根定律把 "或" 转成与非门形式:

A + B·C = \overline{\overline{A}\cdot\overline{B\cdot C}}

所以不需要单独再求一次 B·C 再取反,可以直接利用 G1 的输出 \overline{B\cdot C}:

F = \overline{\overline{A}\cdot\overline{B\cdot C}}

只需再求 \overline{A}(一个非门,与非门两输入接 A),然后将 \overline{A} 与 G1 输出 \overline{B\cdot C} 送入第三个与非门,直接输出就是 F,无需再加末级非门。

电路连接(共 3 个与非门)

B ──┐
    [NAND] G1 ── (BC的反,即 B̄+C̄) ──┐
C ──┘                              │
                                   [NAND] G3 ── F
A ──┬──[NAND] G2 (两输入接A) ── Ā ──┘
    └──┘
  • G1:输入 B、C,输出 \overline{B\cdot C}
  • G2:输入 A、A(同一信号),输出 \overline{A}
  • G3:输入 G2 的输出 \overline{A} 与 G1 的输出 \overline{B\cdot C},输出 \overline{\overline{A}\cdot\overline{BC}} = A + BC = F

答案

$$\boxed{F = A + B\cdot C}$$

用 3 个二输入与非门实现:G1 求 \overline{BC},G2 求 \overline{A},G3 把两者相与再取反即为 F。

易错点

  • 题目原始表达式是 F = A + \overline{A}BC,很多人直接拿这个式子去设计电路,门的数量会变多(多一个 \overline{A} 与 BC 的与门)。一定要先用吸收律化简成 F = A + BC,再设计,门更少更标准。
  • 不要把"A 不合格则需要 B 和 C 均合格"看成"或"(B+C),题目明确写的是"均认为合格",即 B·C(与),不是或。这点容易跟题10的"至少一项"搞混 —— 两题表面相似但一个是"或"一个是"与",审题时要逐字确认。
  • 化简最后一步从"与非-与非"结构看,F 的取值只需 3 个与非门,不需要再加末级非门,因为 \overline{\overline{A}\cdot\overline{BC}} 本身展开后已经等于 A+BC,不要画多余的第四个门。
