题11 JK 触发器画输出波形

题目

电路为 JK 触发器(主从/边沿触发器,具体型号不影响分析方法):

  • 输入端:S(异步置位,接 1,即无效)、1J(即 J)、CP、1K(即 K)、R(异步复位,接 1,即无效)
  • 输出端:Q、Q̄
  • CP 输入端画有小圆圈 → 下降沿触发(每个 CP 脉冲的"下降沿"才是有效触发时刻)
  • 设初始状态 Q=0
  • 给出 CP(1~6 共 6 个脉冲)、J、K 三路波形,要求画出 Q、Q̄ 波形

由于 S=R=1(对这种异步置位/复位端通常低电平有效),所以 S、R 全程不起作用,触发器完全按 JK 同步输入工作。

解答

第一步:确定触发时刻

CP 有圆圈 → 下降沿触发。6 个脉冲对应 6 个下降沿,逐个标记为 CP1↓ ~ CP6↓。

第二步:读取每个下降沿时刻的 J、K 电平

逐沿对照波形图读值:

触发沿 J K JK 组合 触发器动作
CP1↓ 1 1 J=1,K=1 翻转
CP2↓ 0 1 J=0,K=1 置 0
CP3↓ 1 0 J=1,K=0 置 1
CP4↓ 0 0 J=0,K=0 保持
CP5↓ 1 1 J=1,K=1 翻转
CP6↓ 0 0 J=0,K=0 保持

这组 J、K 取值正好把 JK 真值表的 4 种状态(翻转、置0、置1、保持)都过了一遍,是很典型的考法,可以反过来验证读图没读错。

第三步:逐沿推 Q

初始 $Q_0 = 0$

  • CP1↓:J=1,K=1 → 翻转,$Q_1 = \neg Q_0 = \mathbf{1}$
  • CP2↓:J=0,K=1 → 置 0,$Q_2 = \mathbf{0}$
  • CP3↓:J=1,K=0 → 置 1,$Q_3 = \mathbf{1}$
  • CP4↓:J=0,K=0 → 保持,$Q_4 = Q_3 = \mathbf{1}$
  • CP5↓:J=1,K=1 → 翻转,$Q_5 = \neg Q_4 = \mathbf{0}$
  • CP6↓:J=0,K=0 → 保持,$Q_6 = Q_5 = \mathbf{0}$

Q 状态序列(每个 CP 高电平区间内的值,从初态算起):

0(初态) → 1 → 0 → 1 → 1 → 0 → 0

注意:Q 在两个下降沿之间(也就是对应的 CP 高电平及随后的低电平期间)保持不变,只在下降沿那一瞬间根据 JK 跳变;Q̄ 始终是 Q 的反相,全程同步取反即可,不需要单独分析 JK。

输出波形

CP   _┌─┐_┌─┐_┌─┐_┌─┐_┌─┐_┌─┐_____
        ①    ②    ③    ④    ⑤    ⑥
     (CP编号1~6,下降沿①~⑥为有效触发点)

J    _┌───┐___┌───┐___┌───┐___
K    ____┌───────┐_______┌───┐_

       J=1   J=0   J=1   J=0   J=1   J=0
       K=1   K=1   K=0   K=0   K=1   K=0
       翻转   置0   置1   保持   翻转   保持

Q(初态0):
     0  │        1            0  │  1   1  │      0      0
     ───┘                     ┌──┘  └──┐   └──────┐      └────
                               │        │
即:
Q    0__┌────────┐____┌────────────────┐__________________
                       (Q1=1)            (Q2=0,Q3=1,Q4=1)   (Q5=0,Q6=0)

逐段电平(更直观的描述):
  初态 ~CP1↓:        Q=0
  CP1↓ ~CP2↓:        Q=1
  CP2↓ ~CP3↓:        Q=0
  CP3↓ ~CP4↓:        Q=1
  CP4↓ ~CP5↓:        Q=1
  CP5↓ ~CP6↓:        Q=0
  CP6↓ 之后:          Q=0

Q̄ (与 Q 始终反相):
  初态 ~CP1↓:        Q̄=1
  CP1↓ ~CP2↓:        Q̄=0
  CP2↓ ~CP3↓:        Q̄=1
  CP3↓ ~CP4↓:        Q̄=0
  CP4↓ ~CP5↓:        Q̄=0
  CP5↓ ~CP6↓:        Q̄=1
  CP6↓ 之后:          Q̄=1

画图时:Q 的每一段电平要画到对应下降沿处才跳变(不是脉冲上升沿,也不是脉冲中间),Q̄ 整条曲线和 Q 完全对称(高变低、低变高)。

易错点

  1. 触发时刻看错:CP 输入端有小圆圈代表下降沿触发,很多人习惯性当成上升沿触发去读 J、K,导致整组答案全错。看到圆圈先确认触发沿方向再读图。
  2. J、K 取值时刻搞混:JK 触发器是"边沿决定动作",必须读触发沿那一瞬间(本题是每个 CP 脉冲下降的那一刻)J、K 的电平,而不是整个 CP 高电平期间或低电平期间的"大概值"——如果 J/K 在这一瞬间正好处于跳变沿上,要以电路图标注的边沿前一刻为准。
  3. Q 波形跳变位置画错:Q 只能在触发沿(下降沿)处跳变,两次触发沿之间无论 J、K 怎么变都不影响 Q,新手常常把 Q 的跳变画到 J 或 K 变化的位置上,这是错的。
  4. 忘记 S、R 的作用:本题 S=R=1 全程无效,但如果哪天遇到 S 或 R 为低电平有效且被拉低的情况,要先处理异步置位/复位,它优先级高于 J、K,会强制把 Q 置 1 或置 0,而不受 CP、J、K 影响。

一句话总结

下降沿触发的 JK 触发器,只需在每个 CP 脉冲"往下掉"的那一瞬间读 J、K,按真值表(11翻转/01置0/10置1/00保持)逐沿更新 Q,Q̄ 全程与 Q 反相即可。
