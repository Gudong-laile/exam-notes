题 2-25: 三板双剪铆接

图示分析

         ┌─────────────┐
         │   上板  δ   │── F/2 →
         └─────────────┘
         ╔═══════════╗      ← 铆钉, 直径 d
  ← F ──│   中板  δ   │     (穿过三块板)
         ╚═══════════╝
         ┌─────────────┐
         │   下板  δ   │── F/2 →
         └─────────────┘

中板向左拉 F, 上下两板各向右拉 F/2, 力系平衡.

剪切应力 τ — 双剪

铆钉有两个剪切面 (上板/中板之间一个, 中板/下板之间一个), 每个剪切面承担 F/2:

$$\tau = \frac{F/2}{A_{\text{剪}}} = \frac{F/2}{\dfrac{\pi d^2}{4}} = \boxed{\frac{2F}{\pi d^2}}$$

⚠️ 跟单剪的区别: 单剪是 $\tau = \dfrac{4F}{\pi d^2}$, 双剪只有一半. 因为同样的 F 由两个剪切面分担.

挤压应力 σ_bs

中板 (承担全部 F):

$$\sigma_{bs,\text{中}} = \frac{F}{A_{\text{挤}}} = \boxed{\frac{F}{\delta d}}$$

上板/下板 (各承担 F/2):

$$\sigma_{bs,\text{外}} = \frac{F/2}{\delta d} = \frac{F}{2\delta d}$$

最大挤压应力在中板: $\sigma_{bs,\max} = \dfrac{F}{\delta d}$

易错点提醒 ⚠️

  1. 判断剪切面数量靠图: 两块板 = 单剪, 三块板 = 双剪 (中间穿一个铆钉)
  2. 挤压面积是投影面积: $\delta \cdot d$ (板厚乘以铆钉直径), 不是 $\pi d^2/4$
  3. 挤压应力比较: 算中板和外板两个值, 取大的作为校核依据
  4. 力分配要看图: 这道题外板各承担 F/2 是题目隐含的对称, 不是所有双剪都这样

题 5-13: 连续梁边界条件 + 连续条件

图示分析

         q (均布载荷, 向下)
   ↓↓↓↓↓↓↓↓↓↓↓↓↓↓↓↓↓↓↓↓
y  ┌─────────────────────┐
   │                      │  C (固定端)
○──┴────────○─────────────┤
O           B              ┊
↑           ↑              ┊
铰支座      铰支座         ┊
   |←  a   →|←─── l ─────→|
                                → x
  • O ($x = 0$): 滚动铰支座
  • B ($x = a$): 滚动铰支座
  • C ($x = a+l$): 固定端
  • 均布载荷 $q$ 作用在整个 OC 段
  • 这是 1 度静不定连续梁

边界条件 (3 处, 共 4 个方程)

位置约束类型条件
O ($x=0$)滚动铰支座$y(0) = 0$
B ($x=a$)滚动铰支座$y(a) = 0$
C ($x=a+l$)固定端$y(a+l) = 0$, $y'(a+l) = 0$

连续条件 (在 B 处分段, 2 个方程)

由于 B 处是支座 + 分段点, 两侧位移必须连续:

条件方程
挠度连续$y(a^-) = y(a^+)$ (其实因 $y(a)=0$ 已自动满足)
转角连续$y'(a^-) = y'(a^+)$

整理答案 (填到题目空白里)

边界条件: $y(0) = 0$、$y(a) = 0$、$y(a+l) = 0$、$y'(a+l) = 0$;

连续条件: $y'(a^-) = y'(a^+)$ (B 处转角连续)

为什么挠度连续条件被"吃掉"了?

因为 B 是支座, 两边的挠度都必须等于零 ($y(a^-) = y(a^+) = 0$). 所以挠度连续条件已经被 "$y(a) = 0$" 这个边界条件覆盖, 不需要额外写.

只需要写转角连续条件 $y'(a^-) = y'(a^+)$, 因为 B 是铰支座 (允许两侧自由转动), 但分段函数的两侧转角必须在 B 处对接, 这是数学上的连续要求.

求解时的总未知数

  • 2 段梁, 每段 2 个积分常数 → 4 个 $C_i$
  • 1 个多余约束力 (静不定 1 度) → 1 个未知
  • 总共 5 个未知
  • 4 个边界条件 + 1 个连续条件 → 5 个方程

5 = 5, 刚好够解 ✅

易错点提醒 ⚠️

  1. 不要漏 B 处的边界条件 $y(a)=0$ — 中间支座的挠度为零这件事容易忘
  2. 连续条件只需写转角 — 挠度连续被边界条件覆盖了
  3. 如果 B 是内部铰 (不是支座, 只是中间铰链): 那连续条件变成 $y(a^-)=y(a^+)$ + 弯矩为零 $M(a)=0$, 不是这道题的情况
  4. C 是固定端要写两条: 挠度和转角都为零

一句话总结

  • 2-25: 三板双剪, $\tau = \dfrac{2F}{\pi d^2}$ (比单剪小一半), $\sigma_{bs,\max} = \dfrac{F}{\delta d}$ (中板)
  • 5-13: 4 条边界 ($y_O=y_B=y_C=0$, $y'_C=0$) + 1 条连续 ($y'$ 在 B 处连续)
