题 2-25: 三板双剪铆接

图示分析

         ┌─────────────┐
         │   上板  δ   │── F/2 →
         └─────────────┘
         ╔═══════════╗      ← 铆钉, 直径 d
  ← F ──│   中板  δ   │     (穿过三块板)
         ╚═══════════╝
         ┌─────────────┐
         │   下板  δ   │── F/2 →
         └─────────────┘

中板向左拉 F, 上下两板各向右拉 F/2, 力系平衡.

剪切应力 τ — 双剪

铆钉有两个剪切面 (上板/中板之间一个, 中板/下板之间一个), 每个剪切面承担 F/2:

$$\tau = \frac{F/2}{A_{\text{剪}}} = \frac{F/2}{\dfrac{\pi d^2}{4}} = \boxed{\frac{2F}{\pi d^2}}$$

⚠️ 跟单剪的区别: 单剪是 $\tau = \dfrac{4F}{\pi d^2}$, 双剪只有一半. 因为同样的 F 由两个剪切面分担.

挤压应力 σ_bs

中板 (承担全部 F):

$$\sigma_{bs,\text{中}} = \frac{F}{A_{\text{挤}}} = \boxed{\frac{F}{\delta d}}$$

上板/下板 (各承担 F/2):

$$\sigma_{bs,\text{外}} = \frac{F/2}{\delta d} = \frac{F}{2\delta d}$$

最大挤压应力在中板: $\sigma_{bs,\max} = \dfrac{F}{\delta d}$

易错点提醒 ⚠️

  1. 判断剪切面数量靠图: 两块板 = 单剪, 三块板 = 双剪 (中间穿一个铆钉)
  2. 挤压面积是投影面积: $\delta \cdot d$ (板厚乘以铆钉直径), 不是 $\pi d^2/4$
  3. 挤压应力比较: 算中板和外板两个值, 取大的作为校核依据
  4. 力分配要看图: 这道题外板各承担 F/2 是题目隐含的对称, 不是所有双剪都这样

题 5-13: 内部铰梁的边界 + 连续条件

图示分析

B 处的小圆圈 (在梁上方, 不带支座符号) 是内部铰链 (中间铰), 不是滚动铰支座! 这是个常见的图示陷阱.

  • O ($x = 0$): 滚动铰支座
  • B ($x = a$): 内部铰链 — 梁在此处由无摩擦销钉连接, 可独立转动
  • C ($x = a+l$): 固定端
  • OC 段上有均布载荷 $q$

静定性判断

约束反力数
O 滚动铰支座1
C 固定端3
总反力4
方程方程数
静力平衡3
内部铰提供 M(B)=01
总方程4

4 = 4, 静定结构 ✓

边界条件 (3 条)

位置条件
O ($x = 0$)$y(0) = 0$
C ($x = a+l$)$y(a+l) = 0$
C ($x = a+l$)$y'(a+l) = 0$

连续条件 (B 处, 1 条)

只有挠度连续:

$$y(a^-) = y(a^+)$$

⚠️ 转角不连续: 内部铰允许两侧梁独立转动, 所以 $y'(a^-) \neq y'(a^+)$ — 这是内部铰跟支座最大的区别.

总未知数 vs 方程

  • 2 段梁, 每段 2 个积分常数 → 4 个 $C_i$
  • 3 边界条件 + 1 连续条件 = 4 个方程 ✓

跟"中间支座"的对比

情况边界连续
B 是支座 (静不定)$y(0)=y(a)=y(a+l)=0$, $y'(a+l)=0$$y'(a^-)=y'(a^+)$
B 是内部铰 (静定, 本题)$y(0)=y(a+l)=0$, $y'(a+l)=0$$y(a^-)=y(a^+)$ (挠度连续, 转角不续)

易错点 ⚠️

  1. 看清图: 圆圈在梁上方/中间 = 内部铰; 圆圈带三角支撑/底座 = 滚动支座
  2. 内部铰处不写 $y(a)=0$: 内部铰允许 y 在 B 处不等于零, 它只是中间铰链, 不是约束
  3. 转角不连续不要写错: 内部铰处 $y'$ 不连续, 写连续等式就错了
  4. 静定 vs 静不定: 内部铰使原本超约束的结构变成静定, 多个内部铰可以让多跨梁仍然静定

一句话总结

  • 2-25: 三板双剪, $\tau = \dfrac{2F}{\pi d^2}$ (比单剪小一半), $\sigma_{bs,\max} = \dfrac{F}{\delta d}$ (中板)
  • 5-13: B 是内部铰 (静定); 3 条边界 ($y_O=y_C=0$, $y'_C=0$) + 1 条连续 (B 处挠度连续, 转角不连续)
