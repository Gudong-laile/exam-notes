# exam-notes

Engineering exam solutions with LaTeX rendering (KaTeX).

Hosted on GitHub Pages: https://gudong-laile.github.io/exam-notes/

## Structure

- `index.html` — landing page with links to all solutions
- `template.html` — copy this to add a new solution
- `solutions/` — individual solution pages

## Adding a new solution

1. Copy `template.html` to `solutions/your-filename.html`
2. Replace `{{TITLE}}` and `{{CONTENT}}` with actual content
3. Add a link in `index.html`
4. `git add . && git commit -m "Add solution: ..." && git push`
